# Welcome to WasmBridge Organization 🚀

Unified, Sandboxed, and Cloud-Connected Agent Architecture.

WasmBridge is an open-source initiative dedicated to building a secure and scalable ecosystem for remote system management using **WebAssembly**. Our mission is to provide developers with a robust framework to deploy cross-platform agents that are easy to manage, highly secure, and seamlessly connected to the cloud.

## 🌟 Our Vision

We believe that system agents should be:
- **Secure by Default**: Running logic in isolated WASM sandboxes.
- **NAT-Friendly**: Traversing complex networks using outbound "Reverse Push" streams.
- **Modular**: Allowing hot-reloading of functionality without agent restarts.
- **Easy to Use**: Providing intuitive web-based and system-tray interfaces.

## 🛠 Core Projects

- **[WasmBridge](https://github.com/wasmbridge/wasmbridge)**: The main Rust-based agent for Windows.
- **[Plugin Protocol](https://github.com/wasmbridge/plugin-protocol)**: Shared types and communication schemas.
- **[Reverse Push](https://github.com/wasmbridge/reverse-push)**: High-performance bi-directional gRPC communication library.
- **[WasmBridge Plugin Template](https://github.com/wasmbridge/wasmbridge-plugin-template)**: Your starting point for building custom agent logic.

## 💻 Tech Stack

We love modern, safe, and efficient technologies:
- **Languages**: Rust, TypeScript, WebAssembly.
- **Communication**: gRPC, Protocol Buffers, Axum.
- **Runtime**: Extism, Wasmer.
- **UI**: Askama, Vanilla CSS, HTMX.
